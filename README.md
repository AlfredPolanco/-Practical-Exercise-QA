## Installation

1. Clone the repo
```bash
git clone https://github.com/AlfredPolanco/-Practical-Exercise-QA.git
```

2. Install dependencies and packages required for the project
```bash
npm i
```

## Usage

- To run Cypress tests just run the following command
```
npm run tests
```
